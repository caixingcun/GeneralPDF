# GeneralPDF

# MainActivity 通过itext 创建表单 设置内容 生成pdf

# SecondActivity 通过itext 
    获取asset下表单模板pdf 
    通过acrobat添加表单域 
    在代码中遍历表单域填充内容
    生成新的pdf
